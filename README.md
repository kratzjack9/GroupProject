# Life

A Game of Life simulation with custom rules. Organisms have DNA made of 🌱 💧 🌞 🎄 🍊 and live on a grid.

## Rules
- **Hydration** — 💧 organisms spread to empty neighbors.
- **Growth** — 🌱 + 💧 + 🌞 turns into a tree 🎄.
- **Fruiting** — a low-energy tree grows fruit 🍊.
- **Reproduction** — empty cells with the right neighbors birth a new organism.
- **Predator** — organisms with duplicate DNA attack neighbors for energy.
- **Solar Flare** — on cycles 5+ (every 5), 🌞 organisms gain or lose energy.
- **Mutation** — low energy randomly changes DNA.
- **Combust** — all identical DNA self-destructs.
- **Energy** — all organisms lose 1 energy per cycle. 0 = death.
